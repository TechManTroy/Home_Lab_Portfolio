# Home Lab Infrastructure Portfolio
**Maintainer:** Troy Edmonds
**Last Updated:** January 16, 2026
**Version:** 1.0

## Executive Summary
This repository documents the design, implementation, and management of a hybrid home lab environment. The project scope encompasses the migration of containerized workloads to bare-metal infrastructure and the deployment of an enterprise-grade Windows Active Directory (AD) domain.

## Project Phases
* **Phase 1: Infrastructure Migration:** Transitioning Docker containers from VirtualBox to a dedicated Linux host.
* **Phase 2: Enterprise Identity:** Building a Windows Domain Controller, managing users, and automating endpoints via Group Policy.

## Tools & Technologies
* **Virtualization:** VirtualBox 7.0
* **OS:** Ubuntu Server, Windows Server 2022, Windows 11 Enterprise
* **Management:** Docker/Portainer, Active Directory, Jira Service Management
